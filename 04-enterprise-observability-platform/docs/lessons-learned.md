# Lessons Learned

## Technical Lessons

- CloudWatch dashboards provide fast operational visibility.
- Alarms turn metrics into actionable alerts.
- SNS provides a simple notification channel.
- Logs Insights helps investigate application and platform issues.
- CloudTrail provides audit visibility for API activity.
- AWS Config helps detect configuration drift and security risk.
- Systems Manager Automation can standardize remediation.
- Runbooks improve operational consistency.

## Portfolio Lessons

This project is valuable because many cloud portfolios stop after deployment. This project shows the ability to operate, monitor, and support cloud platforms after deployment.

## Real-World Relevance

Cloud engineers are often responsible for triage, monitoring, incident response, audit review, and compliance evidence. This project demonstrates those skills in one cohesive lab.
