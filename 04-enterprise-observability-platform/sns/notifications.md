# SNS Notifications

## Purpose

Amazon SNS sends operational alerts to administrators when CloudWatch alarms enter an alarm state.

## Topic Name

`enterprise-ops-alerts`

## Protocol

Email

## Workflow

CloudWatch Alarm → SNS Topic → Email Notification → Platform Engineer
