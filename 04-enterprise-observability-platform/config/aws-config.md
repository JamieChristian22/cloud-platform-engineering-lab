# AWS Config

## Purpose

AWS Config continuously records resource configurations and evaluates them against compliance rules.

## Configuration Recorder

Recording is enabled for regional resources and supported resource types.

## Delivery Channel

Configuration snapshots are delivered to an S3 bucket.

## Enterprise Value

AWS Config supports:

- Configuration drift detection
- Compliance monitoring
- Security posture review
- Resource inventory
- Change tracking
