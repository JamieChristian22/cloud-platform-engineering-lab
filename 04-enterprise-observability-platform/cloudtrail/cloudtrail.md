# CloudTrail Integration

## Purpose

AWS CloudTrail records API activity across the AWS account. This supports audit readiness, change investigation, and security analysis.

## What to Review

- EC2 events
- IAM events
- CloudWatch events
- Systems Manager events
- Backup events
- Secrets Manager events

## Enterprise Value

CloudTrail answers important questions:

- Who changed this resource?
- When was this API call made?
- What service was accessed?
- Was access denied?
- Did a user or service role perform the action?
