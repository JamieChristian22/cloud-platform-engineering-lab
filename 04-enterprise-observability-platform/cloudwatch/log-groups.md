# CloudWatch Log Groups

## Purpose

CloudWatch Log Groups provide centralized log storage for platform and application events.

## Shared Log Groups

- `/shared/services/application`
- `/shared/services/security`
- `/shared/services/platform`

## Log Retention

Recommended lab retention:

`30 days`

## Enterprise Value

Centralized logging supports:

- Troubleshooting
- Audit support
- Security review
- Incident response
- Operational visibility

## Future Improvements

- Add metric filters.
- Add log-based alarms.
- Forward logs to OpenSearch.
- Add dashboard widgets for log volume.
