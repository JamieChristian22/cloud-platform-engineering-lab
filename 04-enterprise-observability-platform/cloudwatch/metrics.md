# CloudWatch Metrics

## Metrics Tracked

| Metric | Purpose |
|---|---|
| CPUUtilization | Measures EC2 compute load |
| NetworkIn | Tracks inbound network traffic |
| NetworkOut | Tracks outbound network traffic |
| StatusCheckFailed | Detects EC2 health failures |

## Platform KPIs

- Resource health
- Compute utilization
- Network activity
- Alarm state
- Operational risk

## Enterprise Value

Metrics allow teams to detect performance degradation and respond before users are heavily impacted.
