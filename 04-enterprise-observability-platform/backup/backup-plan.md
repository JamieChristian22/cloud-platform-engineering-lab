# Backup Plan Monitoring

## Purpose

Backup plans define backup schedules and retention.

## Related Project

This observability project monitors and documents the backup platform created in the Shared Services Platform project.

`Enterprise-Daily-Backup`

## Operational Checks

- Confirm backup plan exists.
- Confirm assigned resources.
- Confirm backup job success.
- Confirm recovery points are created.
