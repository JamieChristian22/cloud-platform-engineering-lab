# Security Controls

## Identity

- Enforce MFA for administrators.
- Use least privilege roles.
- Avoid root/global admin usage.
- Rotate access keys.
- Review permissions regularly.

## Network

- Deny inbound traffic by default.
- Allow only required ports.
- Segment workloads by trust zone.
- Use private endpoints where possible.

## Data Protection

- Encrypt data at rest.
- Encrypt data in transit.
- Use managed keys where appropriate.
- Apply retention and lifecycle policies.

## Logging and Detection

- Enable administrative activity logs.
- Enable data access logs where needed.
- Alert on privilege escalation.
- Alert on failed login spikes.
- Alert on public exposure changes.

## Compliance Alignment

- CIS Benchmarks
- NIST Cybersecurity Framework
- SOC 2 security principles
- ISO 27001 control themes
