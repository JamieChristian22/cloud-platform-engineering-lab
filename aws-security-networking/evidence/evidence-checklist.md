# Evidence Checklist

Use this folder to store proof that the project was built and validated.

## Required Evidence

- Screenshot of deployed resources
- Screenshot of IAM roles or RBAC
- Screenshot of logging enabled
- Screenshot of monitoring dashboard
- Screenshot of alerts
- Screenshot of encryption settings
- Terraform fmt/validate output
- Architecture diagram PNG
- Cost estimate
- Lessons learned

## Screenshot Naming Standard

```text
01-resource-overview.png
02-networking.png
03-iam-rbac.png
04-logging.png
05-monitoring-dashboard.png
06-alert-rule.png
07-encryption.png
08-cost-estimate.png
```
