# Architecture Decision Record

## ADR-001: Multi-Cloud Portfolio Architecture

### Status

Accepted

### Context

The portfolio needs to demonstrate AWS, Azure, and GCP knowledge while remaining organized enough for recruiters and hiring managers.

### Decision

Use a multi-repository style folder structure with five major technical platforms:

- Cloud Engineering
- Cloud Security Engineering
- DevOps
- SRE
- Cloud Architecture

Each platform contains five job-ready projects.

### Consequences

Positive:

- Shows breadth across multiple cloud domains
- Supports multiple target job titles
- Makes GitHub navigation easier
- Allows projects to be deployed separately

Tradeoffs:

- Requires consistent documentation
- Requires strong evidence capture
- Must avoid looking like empty folders
