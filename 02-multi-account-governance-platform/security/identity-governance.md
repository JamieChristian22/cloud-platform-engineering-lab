# Identity Governance

## Roles
| Role | Responsibility |
|---|---|
| Platform Admins | Organization and platform administration |
| Cloud Engineers | Infrastructure build and operations |
| Security Team | Audit, monitoring, and guardrails |
| Auditors | Read-only review |
| Developers | Workload and sandbox access |

## Principles
- Least privilege
- MFA
- Separation of duties
- Regular access reviews
