# Lessons Learned

## What I Built
- AWS Organization
- Organizational Units
- Service Control Policy
- Governance documentation
- Security model
- Evidence framework

## What I Learned
- AWS Organizations centralizes governance.
- OUs separate responsibility areas.
- SCPs set maximum permissions.
- SCPs do not grant access.
- Governance requires strong documentation.
