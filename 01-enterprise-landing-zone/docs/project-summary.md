# Project Summary

## Project Name

AWS Enterprise Landing Zone

## Repository

cloud-platform-engineering-lab / 01-enterprise-landing-zone

## Summary

Built an enterprise-style AWS landing zone that includes secure networking, identity governance, EC2 workload deployment, logging, monitoring, alerting, and cost governance.

## Key Results

- Deployed VPC with public/private subnet design
- Launched and configured EC2 Linux web server
- Enabled CloudTrail for audit logging
- Created CloudWatch alarm connected to SNS
- Configured AWS Budget for cost control
- Documented implementation and operational procedures

## Skills Demonstrated

- AWS networking
- IAM governance
- EC2 administration
- Linux command line
- Apache configuration
- Monitoring and alerting
- Logging and auditing
- FinOps
- Architecture documentation
